# Vim Editor ( gVim ) Material Theme and Configuration File

Nothing to say. Try using. Let me show you a few basic things.
 - Save using Shortcut Keys.
 - Cut / Copy / Paste.
 - Undo / Redo.
 - Tree View. ( **File Explorer** )
 - Remove Splitter Bars.
 - Autoclosing Brackets. `( ) | " " | ' ' | [ ] | {  }`
 - etc.
